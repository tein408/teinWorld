# teinWorld
Spring Boot Project


## 개발 목적
- Web 백엔드 개발자가 되기 위해 보편적인 기술 중 하나인 Spring boot의 기본기를 다지기 위함입니다.

## 개발 환경
- Java 11
- Spring boot, Spring Data JPA, Spring Security
- H2 database
- gradle
- thymeleaf
- bootstrap 5.0.2
