# teinWorld
Spring Boot Project


## 개발 목적
- Web 백엔드의 보편적인 기술 중 하나인 Spring boot와 Spring Security의 기본기를 다지기 위함입니다.

## 개발 환경
- Java 11
- Spring boot 2.5.3, Spring Data JPA 2.5.3, Spring Security 5.5.1
- H2 database 1.4.200
- gradle 1.0.11
- thymeleaf 3.0.12
- bootstrap 5.0.2
